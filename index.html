Uniform Resource Locators

URL atau web address bisa berupa huruf (ex : www.facebook.com) atau IP address (ex : 198.53.76.32).

Skemanya sebagai berikut : scheme://prefix.domain:port/path/filename
Contoh : https://www.w3schools.com/html/default.asp

  • scheme - define the type of internet service (most common is http or https)
  • prefix - define a domain prefix (default for http is www)
  • domain - define the internet domain name (like facebook.com)
  • port - define the port number at the host (default for http is 80)
  • path - define a path at the server (if ommitted: the root directory of the site)
  • filename - define the name of a document or resource
